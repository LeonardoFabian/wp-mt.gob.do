== Installation ==

1. Upload the entire 'institucionalmt-cpt' folder to the `/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to InstitucionMT CPT Panels menu to customize this plugin.
